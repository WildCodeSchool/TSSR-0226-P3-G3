# Qui est Pharmgreen ?
Son fondateur est de Lyon, et a décidé d’implanter les locaux dans sa ville natale.
La société comprend actuellement 211 personnes réparties dans 11 services (appelés "départements").
Des personnels extérieurs travaillent ponctuellement ou à temps plein avec certains services.
